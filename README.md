## Installation

```bash
#clone repo
git clone https://github.com/agarwalamn/liveboard.git

# change directory
cd liveboard

# install dependencies
npm run install-client
npm run install-server

# run server and client
npm run client
npm run server

#Boom✨ your project is live
```
